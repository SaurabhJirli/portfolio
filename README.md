# Saurabh Jirli — Portfolio

Personal portfolio website built with Next.js 15, React 19, and Tailwind CSS 4.

## About

Portfolio showcasing my work as a Software Development Engineer at Touchcore Systems, Pune. Built with a focus on backend development, distributed systems, and IoT.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS 4
- **Content:** Markdown with gray-matter
- **Language:** TypeScript

## Requirements
- Node.js 18 or 20 (LTS recommended)
- npm

## Commands

| Command                | Action                                           |
| :--------------------- | :----------------------------------------------- |
| `npm install`          | Install dependencies                             |
| `npm run dev`          | Start local dev server                           |
| `npm run build`        | Build production site                            |
| `npm run start`        | Start production server                          |
| `npm run lint`         | Run ESLint                                       |
