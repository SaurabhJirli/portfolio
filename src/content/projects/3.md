## What it does

The site you're looking at right now. A portfolio and blog built from scratch to learn Next.js and practice frontend development outside of my usual backend work.

## How it's built

- **Next.js 15** with the App Router and React Server Components
- **Tailwind CSS v4** for styling — no component library
- **Markdown** files for blog posts, projects, and work entries, parsed with gray-matter and remark
- Dark mode with a blocking theme script to prevent flash
- Client-side search powered by Fuse.js

## Why I built it

I wanted a place to put my work and writing, and I also wanted an excuse to learn React and Next.js properly. Most of my day job is backend Java, so building a frontend project from scratch helped me understand things like server components, routing, and CSS that I'd only seen from the edges before.

## What I learned

Next.js is opinionated in useful ways — file-based routing and server components made a lot of things simpler. Tailwind clicked once I stopped fighting it. The hardest part was getting dark mode right without a flash of white on page load.
